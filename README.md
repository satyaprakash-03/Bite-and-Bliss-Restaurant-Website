# Bite and Bliss 🍽️

A fully responsive restaurant website designed for Bite and Bliss. Features include an interactive menu, table booking, and a contact form.

## 🚀 Tech Stack
- HTML, CSS, JavaScript
- React.js
- Firebase (for backend and hosting)
